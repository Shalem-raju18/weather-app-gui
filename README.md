# 🌦️ Weather App (Python GUI)

A simple and interactive Weather Application built using Python and Tkinter.  
This app fetches real-time weather data using the OpenWeather API.

---

## 🚀 Features

- 🌍 Search weather by city name
- 🌡️ Displays temperature in Celsius
- 🤔 Shows "feels like" temperature
- ☁️ Weather condition description
- 💧 Humidity information
- ❗ Error handling for invalid city

---

## 🛠️ Technologies Used

- Python
- Tkinter (GUI)
- Requests (API handling)
- OpenWeather API

## ▶️ How to Run

1. Install Python
2. Install required library:
   pip install requests

3. Run the application:
   python weather_gui.py

---

## 🔑 API Setup

1. Get API key from OpenWeather:
   https://openweathermap.org/api

2. Replace in code:
   API_KEY = "your_api_key_here"
